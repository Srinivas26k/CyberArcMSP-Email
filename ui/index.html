Provider Setup Guide
📬 Outlook / O365
Go to mysignins.microsoft.com → + Add → App password → copy 16-char key. Admin must enable SMTP AUTH + IMAP.
📧 Gmail
Go to myaccount.google.com/apppasswords → name it "SRV Outreach" → copy 16-char key. Requires 2FA enabled.
⚡ Resend (Recommended for servers/cloud)
Go to resend.com/api-keys → Create API Key → enter it as the "App Password". Key starts with re_. Verify your domain in Resend for best deliverability.
⚠️ Resend accounts cannot check replies (send-only API). Use Outlook/Gmail accounts for reply detection.
